# DOL-language

A lightweight web-based custom script interpreter built with HTML5 + JavaScript.

## Features

- Custom script syntax parsing and execution
- Canvas drawing with customizable brush size and shape
- Loop control with `loopN{cmd}` syntax
- Code save/load and file import/export
- System time query and special effect display
- Easter egg dialogues and light color control
- Variable/array definition and numerical calculation
- Web redirection and multimedia playback
- Profanity word management and special functions
- Quick run with `run` command
